* Name / Value pair: 一個 名稱 / 值 配對代表一個名稱會對應到唯一的值。
* 在一段執行環境(Execution Context)中，一段正在執行的程式碼同樣的名稱(Name)只會有一個，而一個名稱(Name)只能被一個值(Value)定義。
* Object(物件): 名稱 / 值配對的組合。

ex:    
```json
Address: {
	Street: ‘Main’,
	Number: 100,
	Apartment: {
		Floor: 3,
		Number: 301
	}
}
```
